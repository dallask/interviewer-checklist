<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| user input → SearchGroup → Zustand | Search string from input element passed through debounce to setSearchQuery |
| storage-quota-warning CustomEvent → StorageToast | Event dispatched by StorageAdapter; payload not used in UI |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-02-01 | Tampering | SearchGroup search input → DOM rendering | mitigate | Search query rendered via React JSX (auto-escaped text content); never inserted via innerHTML or dangerouslySetInnerHTML — verified by code review |
| T-04-02-02 | Tampering | QuestionCard question/description text from DEFAULT_SECTIONS | accept | DEFAULT_SECTIONS is a build-time constant; not user-supplied data; auto-escaped by React JSX |
| T-04-02-03 | Tampering | StorageToast CustomEvent from StorageAdapter | accept | Event carries only numeric detail (usedBytes, quotaBytes); toast renders only static copy string, no interpolated event data in DOM |
| T-04-02-04 | Denial of Service | ContentTree rendering 1400+ rows | mitigate | useVirtualizer renders only ~20 visible rows; overscan:10; estimated sizes prevent layout thrashing |
| T-04-02-05 | Information Disclosure | aria-live="polite" result count exposes question count | accept | Question bank is a build-time constant (not user PII); exposing count is intentional UX |
</threat_model>
