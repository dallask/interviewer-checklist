<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| bootstrap() uiState → useAppStore.setState | Stored uiState from chrome.storage.local cast to Partial<AppState>; malformed storage could pass unexpected shapes |
| App.tsx buildFlatRows output → ContentTree DOM | Filter results rendered via React JSX; no raw HTML interpolation |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-03-01 | Tampering | main.tsx uiState hydration from storage | mitigate | Spread operator with typed cast `Partial<AppState>`; Sets reconstructed via `new Set(array ?? [])` — malformed non-iterable stored value produces empty Set, not a crash |
| T-04-03-02 | Tampering | App.tsx skip link href="#main-content" | accept | Internal href anchor; no external navigation; no user-supplied href |
| T-04-03-03 | Tampering | Sidebar backdrop click closes sidebar | accept | Closes UI state only; no data mutation; no security consequence |
| T-04-03-04 | Information Disclosure | FOUC reveals dark/light preference before React mounts | mitigate | theme.ts applies OS preference synchronously before React mounts; stored preference applied in async callback; worst case = 1 frame flash on first load, not a security issue |
| T-04-03-SC | Tampering | No new package installs in Wave 3 | accept | No new npm dependencies in this plan; slopcheck not required |
</threat_model>
