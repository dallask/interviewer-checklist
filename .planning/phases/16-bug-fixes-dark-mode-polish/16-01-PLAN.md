<objective>
Fix the note textarea collapse bug (BUG-03) and the score dropdown dark mode contrast issue (POL-01).

Purpose: Two UAT-flagged defects — the note icon fails to close the textarea when content is present (HTML `hidden` attribute bypasses ResizeObserver in React 19 + TanStack Virtual), and the score dropdown options are invisible in dark mode (`<option>` elements cannot be styled via CSS; `color-scheme: dark` is required).

Output: Updated QuestionCard.tsx (textarea className toggle + score select dark class), updated CustomQuestionForm.tsx (difficulty select aligned to standard dark pattern), updated tests in QuestionCard.test.tsx and CustomQuestionForm.test.tsx.
</objective>
