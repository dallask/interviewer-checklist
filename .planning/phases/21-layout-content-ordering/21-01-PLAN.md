<objective>
Implement CONT-01: questions within each topic are displayed in novice → intermediate → advanced → expert difficulty order. Custom questions are merged into the same sorted display at their assigned difficulty position rather than being appended after all default questions.

Purpose: Users see questions in a progressive difficulty ramp within each topic, making it easier to pace an interview from easy warm-up questions to expert-level challenges. Custom questions added by the user slot into that same ramp naturally.

Output: Modified `buildFlatRows.ts` with DIFF_ORDER constant, in-place difficulty sort on filteredQuestions, and a unified merged emit loop replacing the separate default/custom loops. New describe blocks in `buildFlatRows.test.ts` verifying sort order, originalIndex stability, and custom question merge position.
</objective>
