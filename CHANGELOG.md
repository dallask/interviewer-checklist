Initial public release.

- Question bank: 9 groups, ~86 topics, 1000+ questions with 4 difficulty levels
- Difficulty-weighted scoring engine with manual topic overrides
- Per-question and per-topic notes
- Candidate details modal
- Custom questions per topic
- Multiple named sessions with switcher
- YAML export (full structural format) and import (structural + legacy progress-only)
- AI candidate-feedback prompt builder with copy-to-clipboard
- Print stylesheet with full content expansion
- Keyboard shortcuts: / (search), \ (toggle sidebar), Esc (clear/close)
- First-run welcome tab with demo session
- Update banner for version bumps
- Dark mode with OS preference detection
